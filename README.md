# Entertainment-Center
This is the first Udacity Full Stack Web Developer Nano Degree project

The entertainment Center project will provide list of 6 movies on any modern broswer (IE 11 and up or EDGE, FireFox 62 and up and Chrome version 70 and up). Currently it is only designed to run on Windows 7 and up. 

In order to run the Entertainment Center, 
a. You need to download 3 py (Python) files from this site: fresh_tomatoes.py, media.py and entertainment_center.py. 
b. Place these 3 files in one directly. 
c. Make sure Python 3 is installed. In the windows command prompt, type python. Python is installed if you see the following:
    Python 3.7.0 (v3.7.0:1bf9cc5093, Jun 27 2018, 04:06:47) [MSC v.1914 32 bit (Intel)] on win32.
    Type "help", "copyright", "credits" or "license" for more information.
d. If Python has not been installed in your PC, visit www.python.org to download and install Python.
e. To start the Entertainment Center application, open the file folder which has the 3 py files saved. double click entertainment_center.py file. The default web browser will be started and the Entertainment Center application will be loaded. Click each movie icone, it will show the official trailer for the movie.
